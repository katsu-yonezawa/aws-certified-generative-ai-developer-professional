# Temperature とは何か

LLM における temperature は、次に出す単語の候補の確率分布を、どれくらい鋭くするか、あるいは平らにするかを調整するパラメータです。

## 名前の由来

この temperature という名前は、統計力学の温度に由来します。
物理では、温度が高いほど粒子の状態分布が広がり、温度が低いほど低いエネルギー状態に集中しやすくなります。

LLM でも似た考え方で、temperature を使って単語候補の分布の広がりを調整します。

- 温度が低い
  確率の高い候補に強く集中する
- 温度が高い
  候補が広く選ばれやすくなる

つまり、分子の運動そのものを計算しているわけではありませんが、確率分布の広がり方を表す考え方として、物理の温度のイメージが借りられています。

## 何が起きているのか

モデルは、次の単語候補ごとに点数のようなものを内部で持っています。
この点数から確率を作るときに、temperature が使われます。

直感的には次のように考えると分かりやすいです。

- temperature が低いと、上位候補がより有力になる
- temperature が高いと、下位候補にも確率が回りやすくなる

そのため、temperature は出力の安定性と多様性に影響します。

## 挙動のイメージ

### 低い temperature

- 出力が安定しやすい
- 毎回かなり似た答えになりやすい
- 無難で保守的な応答になりやすい

### 高い temperature

- 出力のばらつきが大きくなる
- 意外な表現や発想が出やすい
- 一貫性が下がることがある

## よくある誤解

temperature は hallucination を直接なくすための仕組みではありません。

- 低くすると答えは安定しやすい
- しかし、安定して間違うこともある

したがって、temperature は主にランダムさや多様性を調整するためのものであり、事実性の検証そのものとは役割が違います。

## まとめ

temperature とは、LLM の出力候補の確率分布の広がり方を調整するパラメータです。
名前は統計力学の温度に由来しており、

- 低温では有力候補に集中する
- 高温では候補が広く分散する

という対応があります。

そのため、temperature は

- 安定した回答を出したいのか
- 多様で発想的な回答を出したいのか

を調整するための値だと理解すると整理しやすいです。

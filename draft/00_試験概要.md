# AWS Certified Generative AI Developer - Professional 試験概要

## ひとことで

AWS 上で生成 AI を本番実装する開発者向けの上位試験。  
単なるモデル理解ではなく、`FM統合 + RAG/ベクトル検索 + エージェント + セキュリティ/ガバナンス + 運用最適化 + 評価/障害対応` を実務レベルで問う。

## この試験が証明すること

- 基盤モデル (FM) をアプリケーションや業務フローに統合できる
- ベクトルストア、RAG、ナレッジベースを使った GenAI アーキテクチャを設計・実装できる
- プロンプトエンジニアリングとプロンプト管理を実装できる
- エージェンティック AI を実装できる
- コスト、性能、ビジネス価値を最適化できる
- セキュリティ、ガバナンス、Responsible AI を実装できる
- 評価、監視、トラブルシューティングまで含めて運用できる

## 想定受験者

- AWS またはオープンソースで本番アプリを構築した経験が 2 年以上
- 一般的な AI/ML またはデータエンジニアリングの経験がある
- GenAI ソリューション実装の実務経験が 1 年ある

推奨される前提:

- AWS のコンピューティング、ストレージ、ネットワーク
- セキュリティベストプラクティスと IAM
- デプロイ、IaC
- 監視、オブザーバビリティ
- コスト最適化

## 対象外

- モデル開発とトレーニング
- 高度な ML 手法
- データエンジニアリングと特徴量エンジニアリング

要点: この試験は「モデルを作る人」より、「既存の FM を安全に本番組み込みする人」向け。

## 試験形式

試験ガイド (`AIP-C01`) ベース:

- 択一選択問題、複数選択問題
- 採点対象 65 問
- 採点対象外 10 問
- スコアは 100-1000
- 合格スコアは 750
- セクションごとの足切りではなく、総合点で判定
- 未回答は不正解
- 推測による減点なし

注記:

- 2026-03-15 時点で、AWS 認定ページには `Beta registration is open`、`205 minutes`、`85 questions`、`150 USD`、`English and Japanese` という表記がある
- 一方で、試験ガイドは `AIP-C01` の本試験構成として上記の採点ルールを案内している
- 公式ページ間で表記差分があるため、申込前に AWS Training / Pearson VUE の予約画面で最新条件を再確認する

## 出題ドメイン

### 1. 基盤モデルの統合、データ管理、コンプライアンス `31%`

最重要ドメイン。学習の中心。

- 要件に合う FM 選定
- データ検証、前処理、マルチモーダル入力
- ベクトルストア設計
- 検索機構、埋め込み、チャンク分割、RAG
- プロンプト設計、Prompt Management、ガバナンス

### 2. 実装と統合 `26%`

- エージェンティック AI とツール連携
- FM デプロイ戦略
- 既存業務システムとの統合
- FM API 統合、ストリーミング、リトライ、ルーティング
- API Gateway、Lambda、Step Functions などを使った実装パターン

### 3. AI の安全性、セキュリティ、ガバナンス `20%`

- 入出力の安全制御
- ハルシネーション低減
- プロンプトインジェクションや不正入力対策
- データ保護、PII、プライバシー
- IAM、VPC、監査、ポリシー、Responsible AI

### 4. GenAI アプリケーションの運用効率と最適化 `12%`

- トークン効率とコスト最適化
- レイテンシ最適化
- キャッシュ
- 性能改善
- 監視、可観測性、メトリクス設計

### 5. テスト、検証、トラブルシューティング `11%`

- FM 出力の評価
- RAG 評価
- A/B テスト、継続評価、品質ゲート
- 障害解析
- プロンプト、検索、統合の不具合切り分け

## 学習の起点

- まずはドメイン 1 と 2 を固める
- 次にドメイン 3 を重ねて、安全性と統制を実装観点で理解する
- ドメイン 4 と 5 は「本番運用でどう壊れ、どう測り、どう改善するか」で覚える

頻出の軸:

- `どの FM / 構成を選ぶか`
- `RAG をどう作るか`
- `Agent をどう安全に動かすか`
- `品質、コスト、レイテンシのトレードオフをどう取るか`
- `本番監視と評価をどう回すか`

## まず覚えるキーワード

- Amazon Bedrock
- Knowledge Bases
- Guardrails
- Prompt Management
- ベクトルストア、埋め込み、チャンク分割、再ランキング
- Agents、tool use
- IAM、VPC、CloudWatch、CloudTrail
- API Gateway、Lambda、Step Functions
- 評価、回帰テスト

- この試験は何を証明する試験か
